# joonas.fi's dotfiles

My set of dotfiles.

## Installation

	# Clone the repo
	hg clone https://joonas_fi@bitbucket.org/joonas_fi/dotfiles ~/.dotfiles
	cd ~/.dotfiles

	# Set up symlinks
	./install.sh

	# Go back to where I was
	if [[ $OLDPWD ]]; then cd $OLDPWD; fi
